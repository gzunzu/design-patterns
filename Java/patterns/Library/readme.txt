Ejemplo de uso de LinkedList: Adición, lectura y borrado.
Uso de herencia simple.
Adicionalmente, se trabaja con lectura y escritura en fichero de texto.

El programa ofrece un menú que permite añadir, eliminar y visualizar objetos –Libro o Cómic– de una colección LinkedList.
Una de las opciones de dicho menú permite escribir datos de la colección en un fichero, y leer el resultado.
