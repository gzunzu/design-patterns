Ejemplo de uso de inyección de dependencias. Uso puntual de una interface.

El programa contiene una clase que es capaz de gestionar órdenes –pedidos–, con independencia del tipo de canal por el que hayan sido recibidas –teléfono, web, tienda–, aun teniendo en cuenta las particularidades funcionales de cada uno: número de teléfono, IP del cliente, etc.
